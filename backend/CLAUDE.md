# Backend Development Guidelines

## Tech Stack
- **Language**: Rust (stable, edition 2024)
- **Framework**: Axum (web framework + WebSocket)
- **Async runtime**: Tokio
- **Linting / Formatting**: Clippy + rustfmt

## Commands

```bash
cargo test
cargo clippy -- -D warnings
cargo fmt --check
```

## Code Style

Follow standard rustfmt + Clippy conventions.
